# 3D-objects-with-OpenGL-and-GLCube
Simple examples of 3D objects OpenGL and GLCube.
Requires Pygame, OpenGl, and GlCube.

Example image: <img width="722" height="495" alt="Screenshot 2026-05-12 113909" src="https://github.com/user-attachments/assets/14a0edb8-1292-4c22-96bd-6b3849258541" />

How to install in the terminal:

pygame: `pip install pygame`

GLCube: `pip install pyopengl numpy`

OpenGL: `pip install PyOpenGL PyOpenGL_accelerate pygame numpy`

NOTE: I use Windows and `pip` is only the Windows package manager if you use something else then make sure to replace `pip` with whatever you use.
